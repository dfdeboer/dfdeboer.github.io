Embedded Microcontroller Systems
A junior-level course required for engineering majors in the electrical and computer concentrations. It is an optional course for those in the biomedical concentration. This
course was offered every spring semester from 2008 through 2017,
then again in the spring of 2020.
It is planned to be offered in each spring semester of even calendar years starting in 2020.

Lab 5, Raspberry Pi and interrupt driven I/O (3/24)

Instructions to students:
Here is a source of information from RasPi.Tutorials, Videos, and Reviews The goal of this lab is to establish a tic-clock that generates an interrupt once each 1/60'th of a second. Use this interrupt to cause an LED to blink once each second. (0.5 seconds "on," 0.5 seconds "off," and repeat forever) while the LED is blinking your code should run some other program, for example "railroad crossing" or "cylon eyes." The other code should not use the interrupt system. A schematic for the hardware needed to generate the tic clock can be found here.

Lab 6 Project (3/31, 4/07, 4/14, 4/21, 4/28)

Instructions to students:
Create a program that uses both digital input and digital output.
The program must run automatically on boot-up without an attached keyboard or monitor.
The program must have a useful hardware I/O interface such as switches, LEDs, etc. (Use the GPIO port.)
Example: A stop watch, with set, clear, lap buttons and a 7-segment LED display.
Deliver a report on your project.
Student Work

Spring 2016 Lab, Taught by Professor De Boer,
with Prof. Sara Alsum-Wassenaar from the Art department
This semester the lab was turned into a cooperative project between the Art Department and the Engineering Department. The goal was to create technology-based art. After a couple of ungraded introductory labs the EGR 304 students began working alongside art students on a project. The engineering students were responsible for the technical aspects of a technology-based artistic endeavor.

The first three labs were not graded
Lab 1, Blind-Cycle I/O on the Arduino (1/18)

Lab 2, Serial Interfacing (1/25, 2/01)

Lab 3, Transmission Lines (2/08)

Lab 4, Project (the rest of the semester)
