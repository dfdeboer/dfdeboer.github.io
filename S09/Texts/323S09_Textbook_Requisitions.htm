Comments:

No new sales are likely since this is a continuation from last semester. Used copies should have a CD, although this is not required. The book comes with a CD. I don't think any 5th edition copies of this text are sold (by the publisher) without the CD. The CD contains nothing that cannot be downloaded for free from the web except for a 52 page supplement to the textbook (a single PDF file). I will not be using the supplement for the course, although students will prefer to have it if at no extra cost. There is a "5th edition" ISBN 0-19-514251-9 and a "Revised 5th edition" ISBN 978-0-19-533883-6. The "Revised 5th edition" includes corrections to some (but by no means all) typographic errros that I know of in the "5th edition" and new advertising on the front inside covers. As far as I can tell, the advertising is the only substantive difference. Thus I don't care which version of the 5th edition the bookstore stocks, except for a slight preference for the "revised 5th edition" since some typos have beem corrected in that version.
