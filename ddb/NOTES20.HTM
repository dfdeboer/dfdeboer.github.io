Notes on DDB's web pages

To be sure your browser is giving you a view of the most recently posted page, flush the current page from your cache.
Windows, press Ctrl-F5
On a Windows computer, press and hold down the Ctrl key, then press F5, release F5 and then release the Ctrl key. Do this after loading a suspected out-of-date page.

Apple, press Cmd-R
On an Apple computer, press and hold down the Cmd key, then press R, release R and then release the Cmd key. Do this after loading a suspected out-of-date page.

Note 1: On some web pages you need to press Ctrl-F5 or Cmd-R up to three times in a row to make it work! In these cases the first keystroke (Ctrl-F5 or Cmd-R) will merely reload the page from cache. You need to make the second and third keystrokes before the browser finishes reloading from the cache in order to force the browser to flush the cache and reload via the Web from the file server. This will flush only the page you are currently viewing from your cache, replacing that page in your cache with fresh information. All other cached page functions are unchanged.

Note 2: To get back to the page you came from, use the "back" feature of your browser. (Click the icon with a left-pointing arrow on it or try alt-leftarrow on the keyboard.)

Note 3: Of course the issue of cached Web pages is actually more complicated. See more details here.
