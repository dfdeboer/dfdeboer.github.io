Tektronix TDS 200 – Series

Digital Real Time Oscilloscope

 

Instructions and Safety

 

The Oscilloscope is used as a real time voltmeter to visually access and measure voltage signals.There are two manuals for the Tektronix Digital Oscilloscope, The User Manual and the programmer manual.  Also, there is a Oscilloscope Primer The XYZs Of Using A Scope published by Tektronix.  The Primer provides basic information on how a scope works and its capabilities.  The Users Manual provides some background information but focuses mainly on using the features specific to the Tektronix Digital Scope.  The Programmer manual provides information about operating the Scope using an RS-232 or GPIB interface.

 

 

General Specs:

-         60 MHz bandwidth

-         1 GS/s sample rate and 2,500 point record length for each channel.

-         Accuracy: about +/- 4% x (reading + 0.1 div + 1 mV).  See manual for more

     detailed specifications.

-         Sec / Div Range: 5ns/div to 5 s/div, in a 1, 2.5, 5 sequence.

-         8 bit resolution digitizers, each channel sampled simultaneously.

-         Overvoltage: Cat 1 and Cat 2: 300 Vrms, 500 Vpeak.  Cat 3: 150 Vrms

 

Basic safety:

 -Do not disconnect probes from scope while they are connected to a voltage    source.

-Observe all terminal ratings to avoid overloading.

-Keep away from exposed circuitry.

- Do not exceed 300 Vrms, 500 Vpeak.
