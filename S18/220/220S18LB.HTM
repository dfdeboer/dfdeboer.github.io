Resistor Color Code Calculator
The above link works for resistors having 3 or more color bands, BUT if the resistor has only 3 or 4 bands, you must leave the FIRST dialog BOX in the calculator BLANK or you will get wrong results. For example, brown, black, red (only three bands) is 1K ohm and 20% tolerance. Also, brown black yellow gold is 100 K ohms and 5% tolerance. (Try those to be sure you are getting good results.)
