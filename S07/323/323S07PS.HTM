Note 1) The text includes "Exercises" embedded throughout the 
reading assignments and "Problems" at the end of each chapter.  
All the assignments refer to "Problems" unless they are prefixed 
with "Ex".  Example:  Ex1.5 is found on page 10 but 1.5 is found 
on page 51.

Note 2) A "+" mark in front of the problem number means an answer 
is in the text, in Appendix H.

Note 3) The letter "D" in front of a problem number indicates a 
design oriented problem.  The student may need to use some 
judgment in understanding what the problem statement is 
requesting.  (Ask Prof. De Boer if you are in doubt.)

Note 4) Asterisks (*) indicate more difficult or more time-
consuming problems.  The more asterisks, the more difficult.

Note 5) If a "#" follows the "returned" date, then the solution
key is posted on the cork board across the hall from room S233.
Solution keys will usually be posted shortly after homework is 
returned. 

Note 6) Problems sets shown above with no "assigned" date are 
tentative.  More problems might be added, expected due dates 
might change, but problems shown will eventually be assigned.
