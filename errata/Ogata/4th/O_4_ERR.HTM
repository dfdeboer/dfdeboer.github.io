TEXTBOOK ERRATA

Ogata, Katsuhiko
Modern Control Engineering, 4th Ed., Prentice Hall, 2002.
ISBN 0-13-060907-2

(This list was started on 8/12/2008. Last update was 9/05/08)

Page 52. Problem B-2-20.
Add the following to the end of the problem statement:
a.) Assume 0 < ζ < 1
b.) Assume ζ = 1
c.) Assume ζ > 1           (updated 9/05/2008)

Page 52. Problem B-2-23.
Change, "foreing" to, "forcing"
(The last sentence means that x and all its derivatives are zero for t < 0.)
